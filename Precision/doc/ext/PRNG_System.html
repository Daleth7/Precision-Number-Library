Pseudo-Random Number Generator System

The pseudo-random number generator system of the library generates numbers of arbitrary size. The default instantiation ( Precision::Random) provides a functor for generating psuedo-random integers.
In addition to the default generator, there is a PRNG designed to accommodate general base numbers.
