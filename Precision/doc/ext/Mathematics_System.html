Mathematics System (Math namespace)

A group of mathematics methods under the Math namespace to perform additional operations on the number types provided by the Precision Number Library. All of the methods provided by this system shall also support any number base. The mathematics system can be accessed by including Precision/Precision_Math.h.
Note: Although the majority of the methods provided by this namespace are templates, the user need only worry about passing in appropriately typed parameters.

Associated File: Precision/Precision_Math.h

This system is divided into the following subsystems:

  • Integral subsystem
  • Floating point and fractional subsystem
  • Trigonometric subsystem
The following are provided in the Math namespace.
