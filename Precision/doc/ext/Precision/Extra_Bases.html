Extra Bases

Associated File: Precision/Extra_Bases.h

The Precision Number Library provides not just the decimal instantiation, but also instantiations for the following bases:

  • Binary
  • Octal
  • Hexadecimal
  • Base 36
  • ABC

  • Class Interface Information
  • Example Use
    • This is only of the make functions. Use the links to each number base for more examples.
Specifically, the Binary base is a specialization that is memory efficient at the cost of speed (though the bitwise operations are actually faster). It also includes a couple more member types and additional shifting operations. More information about this specialization can be found here.

Additionally, make functions are available to create one of the number types using default template parameters. These make functions are only for bases 2 to 36.

To access these functions, define USE_BASE_MAKER_ before including the header.
Warning: The current implementation takes a few seconds longer to compile, hence the need to define a macro in order to use it.
To call of these functions, use the following format, replacing the contents in square brackets with the desired parameters): "make_[Number Type][Number Base]([Either an integer or a string])".
Example calls would be:
make_int7(1000);
make_float14("AB45.C03");
