Hexadecimal Instantiation

A base 16 set of number types provided by the Precision Number Library. It includes the types:

  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract


Associated File: Precision/Precision_Hexadecimal.h
For detailed information of the interface, refer to:
  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract


Example Use
  1. #include "Precision.h"
  3. #include <iostream>
  5. int main(){
  6.     using namespace Precision;
  7.     using namespace std;
  9.     Hexadecimal::Int i1(-1125), i2("2A5");
  11.     cout << i1.str() << '\n' << i2.str() << '\n'; //Displays -465 and +2A5
