Decimal (Default) Instantiation

Associated File: Precision/Precision_Default.h

The default set of number types provided by the Precision Number Library. It includes the types:

  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract
All of which are in base 10 and use char to represent each digit, e.g. "10022023456789".
For detailed information of the interface, refer to:
  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract

In addition to the types listed above, the default instantiation comes with a set of literal operators for the integral and floating point types. For more information of the literal operators, click here.
