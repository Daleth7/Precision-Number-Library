Tag Subsystem

This subsystem contains tags that denote which category of numbers a particular number type belongs to. For example, the type Precision::Int belongs to the Signed and Integral categories.

Associated File: Precision/General_Base/Impl/Precision_Tags.h

  • Class descriptions
  • Example Use

Namespace Class Description
Tag Integral The number increments and decrements by exactly 1. It is always whole and has no fractional portion.
Floating_Point The number is not whole and has a fractional portion. It also contains a certain precision to indicate how many digits past the point the number may contain. Floating_Point numbers shall be able to be constructed from and be convertible to Integral numbers.
Fraction The number contains both a numerator and denominator, both of which are integral. The number represents a non-whole number with a fractional portion and a certain precision. Fraction numbers shall be able to be constructed from and be convertible to Integral and Floating numbers.
Complex The number contains both a real number and an imaginary number. The coefficients of these numbers are real. This number follows the format a+bi where a is the coefficient of the real part and b is the coefficient of the imaginary part. Complex numbers shall be convertible to Polar numbers.
Polar The number contains both a magnitude and an imaginary number. The coefficients of these numbers are real. This number follows the format M*e(Pi) where M is the magnitude and P is the phase. Polar numbers shall be convertible to Complex numbers.
Unsigned The number has no sign component as the sign is always positive. The minimum value of the number is guaranteed to be 0.
Signed The number has a sign component to indicate whether it is positive or negative. By convention, numbers in this library shall follow the following rules:
  • short(1) == positive
  • short(-1) == negative
Set The number is not a single number but rather a collection of numbers. Regular arithmetic operations will not apply to Sets, but Sets will allow access to individual elements.
