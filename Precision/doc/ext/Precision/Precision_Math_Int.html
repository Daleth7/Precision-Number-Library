Integral Mathematics Subsystem (template)

These operations are reserved for integral types only. Their behaviour is not guaranteed should a non-integral number be passed to them.

Namespace Class Category Member
Math Functions
Integer_Type factorial(const Integer_Type& start)
Calculate the factorial of the parameter, which is defined to be:
start! = (start-0)*(start-1)*(start-2)*...*(1)
To find the factorial of a floating point number, use factorial_gamma(const Number_Type&).

Integer_Type factorial_range(const Integer_Type& start, const Integer_Type& end)
Calculate the the product of a range, defined as:
f(start, end) = (start+0)*(start+1)*(start+2)* ... *(end-2)*(end-1)*(end-0)

Integer_Type gcd(const Integer_Type& f, const Integer_Type& s)
Calculate the Greatest Common Divisor, or the Greatest Common Factor of the two parameters.

Integer_Type lcm(const Integer_Type& f, const Integer_Type& s)
Calculate the Least Common Multiple.

Integer_Type fibonacci(std::size_t* term_request = nullptr, std::size_t starting_term = 1)
Retrieve the next number in the Fibonacci sequence. The sequence itself is stored within the function.
term_request
An optional request to retrieve which term the sequence is currently on.
starting_term
A number aside from the default is a request to restart the sequence at the passed starting_term.
Warning: This function is due to be replaced by a Precision::Sequence factory method.

void erato_sieve(OutIter dest, std::size_t max)
Construct a set of prime numbers and fill up a container with those numbers.
dest
A forward iterator of a container large enough to store the prime numbers (it is recommended to use std::insert_iterator or the equivalent if the size is unknown). The prime numbers are placed in the container through this parameter.
max
The maximum number possible in the set to generate. For example, if max is 10, the generated set will be [2 3 5 7].

Warning: This function is due to be replaced by a Precision::Sequence factory method.
