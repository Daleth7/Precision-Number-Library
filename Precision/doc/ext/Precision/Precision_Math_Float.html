Floating Point and Fractional Mathematics Subsystem (template)

These operations will work with most of the number types from the Precision Number Library. As part of the Precision Number Library, these methods will also work for any number base.
Most of the listed methods should work with integral types as well.

Warning: Most of these functions are either currently under construction or still need to be tested and fixed.
