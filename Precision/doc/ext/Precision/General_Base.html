Precision::General_Base namespace

The General_Base namespace contains all the class templates for instantiating the various, generic types of the Precision Number Library. To include the namespace, simply include Precision/General_Base.h.

The templates provided in the General_Base namespace include:

  • Precision::General_Base::Int
  • Precision::General_Base::Float
  • Precision::General_Base::Fract
  • Precision::General_Base::UInt
  • Precision::General_Base::UFloat
  • Precision::General_Base::UFract
  • Precision::General_Base::Complex (Under Construction)
  • Precision::General_Base::Random
