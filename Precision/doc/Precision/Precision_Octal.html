Octal Instantiation

Associated File: Precision/Precision_Octal.h

A base 8 set of number types provided by the Precision Number Library. It includes the types:

  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract

This number base may be accessed through the Octal namespace. For detailed information of the interface, refer to:
  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract
