Decimal Literal Operators

Associated File: Precision/General_Base/Precision_Literal_Operators.h

For the decimal instantiation, there is a set of literal operators in the global space. They allow the client to construct objects with the usual parameters, i.e. integer/floating point values and string objects.

  • Function Information
  • Example Use


Precision::Int
Int operator"" _p_int(char const *const, std::size_t)
Construct a Precision::Int object from a string. E.g. "123"_p_int

Int operator"" _p_int(unsigned long long)
Construct a Precision::Int object from an unsigned integer value. E.g. 237864_p_int
Note: Remember that, even if the integer is prefixed with a negative sign, the sign is parsed as a lower precedent operator, i.e. writing -123_P_Int is equivalent to -(123_p_int).
Precision::Float
Float operator"" _p_float(char const *const, std::size_t)
Construct a Precision::Float object from a string. E.g. "123.456"_p_float

Float operator"" _p_float(unsigned long long)
Construct a Precision::Float object from an unsigned integer value. E.g. 237864_p_float

Float operator"" _p_float(long double)
Construct a Precision::Float object from an signed, floating point value. E.g. -237.864_p_float
