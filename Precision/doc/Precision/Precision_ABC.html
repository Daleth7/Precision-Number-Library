ABC Instantiation

Associated File: Precision/Precision_ABC.h

A base 26 set of number types provided by the Precision Number Library. It includes the types:

  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract

It uses the uppercase version of the modern English alphabet, i.e. letters "A" as 0 to "Z" as 25.
This number base may be accessed through the ABC namespace. For detailed information of the interface, refer to:
  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract
