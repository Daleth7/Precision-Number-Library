Trigonometric Mathematics Subsystem (template)

These provide trigonometric operations for the Precision Number Library. These support any number base and should work with most number types.

Note: All trigonometric operations provided will treat the parameter as if it were in radians or will return the answer in radians.

2nd Note: Most of these operations will allow the user to pass in a constant value that is equivalent to the mathematical pi. This may be necessary if a different precision of pi is needed or if Number_Type cannot be constructed from k_pi_str, a C-string.

Warning: All of these functions are either currently under construction or still need to be tested and fixed.
