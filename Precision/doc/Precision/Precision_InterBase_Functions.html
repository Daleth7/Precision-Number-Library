Inter-Base Functions

Associated File: Precision/Precision_Literal_Operators.h

In addition to supporting any radix, these functions provide operations between two or more number bases.

  • Function Information
  • Example Use

In the following list of functions, the template parameters are described and marked with either explicit or deduced. These are as follows:
  • explicit - This parameter must be specified.
  • deduced - This parameter is deduced based on the type of the function argument. There is no need to specify this parameter explicitly.


Template Parameters Functions
Base_Return (explicit)
The number base to convert to.
Base_Param (deduced)
The number base to convert from.
Base_Return convert(const Base_Param&)
Convert the parameter from its current base (Base_Param::base()) to the specified base (Base_Return::base()). This function currently only works for integral types.

Example Use
  1. #include "Precision.h"
  3. #include <iostream>
  5. int main(){
  6.     Precision::Int dec(777); //Base 10
  7.     Precision::Hexadecimal::Int hex; //Base 16
  9.     // Convert to base 16
  10.     hex = Precision::convert<Precision::Hexadecimal::Int>(dec);
  12.     std::cout
  13.         << dec.str() << '\n' // +777
  14.         << hex.str() << '\n' // +309
  15.     ;
  17.     return 0;
  18. }
