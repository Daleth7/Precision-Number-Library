Base 36 Instantiation

Associated File: Precision/Precision_Base36.h

A base 36 set of number types provided by the Precision Number Library. It includes the types:

  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract

This number base may be accessed through the Base36 namespace. For detailed information of the interface, refer to:
  • Int
  • Float
  • Fract
  • UInt
  • UFloat
  • UFract
